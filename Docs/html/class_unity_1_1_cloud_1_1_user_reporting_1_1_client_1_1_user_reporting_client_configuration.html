Parameters
maximumEventCountThe maximum event count. This is a rolling window.
maximumMeasureCountThe maximum measure count. This is a rolling window.
framesPerMeasureThe number of frames per measure. A user report is only created on the boundary between measures. A large number of frames per measure will increase user report creation time by this number of frames in the worst case.
maximumScreenshotCountThe maximum screenshot count. This is a rolling window.
